## Book Mall Management System

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (any of the listed software can be used)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Functional Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Registration
3. Homepage
4. Books
5. Book Total Popularity Ranking
6. Personal Center, Balance Recharge
7. Backend Management
8. Shopping Cart
9. Delivery Address
10. Book Favorites
11. Book Orders


**Administrator Functions:**
1. Login
2. Personal Center (Change Password, Personal Information)
3. Basic Data Management (Member Level Type Management, Book Type Management)
4. Book Management (Book Management, Book Review Management, Book Favorites Management, Book Order Management)
5. User Management
6. Carousel Information
